  # 体感控制器

  _✨ 梦幻体感控制器是一个基于人工智能视觉识别的动作捕捉系统。 ✨_

  通过摄像头实时捕捉用户动作，结合云平台的动作配置，将识别到的动作转换为键盘按键输入，从而实现体感游戏控制。
